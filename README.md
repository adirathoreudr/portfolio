# ADITYA RATHORE — The Developer Issue, № 001

**Live: [adirathoreportfolio.vercel.app](https://adirathoreportfolio.vercel.app)**

An editorial-magazine portfolio. Deep crimson, pitch black, stark white.
Playfair Display for the drama, Inter for the facts.

## Stack

- Next.js (App Router) · TypeScript
- Tailwind CSS v4
- Framer Motion · Lenis smooth scrolling
- HTML5 Canvas (the arcade lives in the footer)

## Run it

```bash
npm install
npm run dev
```

## Anatomy

| Section | What happens there |
| --- | --- |
| The Masthead | A cover, not a hero. |
| The Marquee | Certifications on an endless loop. |
| The Features | Four projects, admitted into evidence. |
| The Record | A live commit garden and the arsenal. |
| The Epilogue | Book a call. Find the terminal. Beat the bird. |

Contribution data is fetched live and revalidated hourly; when the wire
goes quiet, the garden is reconstructed from memory.

The first rule of good infrastructure — it never speaks of itself.
